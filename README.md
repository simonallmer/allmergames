# Futory Cards

## Overview
Futory Cards is an innovative card game that combines strategy, luck, and fun! Play with friends and family in various game modes.

## Getting Started
To set up your project, follow these steps:

1. Clone the repository: `git clone https://github.com/simonallmer/Futory-Cards.git`
2. Navigate to the project directory: `cd Futory-Cards`
3. Follow the instructions in the documentation to install dependencies and start playing!

## Game Rules
1. Each player starts with a set of cards.
2. Players take turns to draw cards and execute actions based on the card type.
3. The goal is to reduce your opponent's score to zero.
